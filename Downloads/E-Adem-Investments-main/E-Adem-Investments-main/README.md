# E-Adem Investments

A professional microfinance company website built with modern web technologies.

## Overview

E-Adem Investments is a fintech company providing:
- Micro Loans (₦30,000 - ₦500,000)
- Savings & Investment Services
- Forex Trading Education

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styles with Tailwind CSS utilities
- **JavaScript** - Vanilla JS for interactivity
- **Tailwind CSS** - Utility-first CSS framework

## Features

- Fully responsive design
- Modern, professional UI with consistent color palette
- Smooth animations and transitions
- Mobile-friendly navigation
- Testimonial carousel
- Counter animations
- Scroll-triggered animations
- Contact form with validation

## Pages

- **Home** - Hero section, services overview, statistics
- **About** - Company story, values, mission
- **Services** - Loan products, savings, forex education
- **Team** - Leadership team profiles
- **Ambassador** - Brand ambassador OYEDIJI OMOSHALEWA ELIZABETH
- **Testimonials** - Client success stories
- **Market Insights** - Industry analysis
- **CEO Mindset** - Leadership philosophy
- **Contact** - Contact form and information

## Color Palette

- **Primary Blue**: #1e40af (navy-700)
- **Primary Light**: #3b82f6 (primary-500)
- **Navy Dark**: #0f172a (navy-900)
- **Gold Accent**: #d97706 (gold-600)
